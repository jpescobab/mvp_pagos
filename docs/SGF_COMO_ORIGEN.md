# SGF como fuente de origen

SGF entrega datos y documentos de origen. El sistema conserva la evidencia recibida, pero no adopta los estados, grupos ni responsables de SGF como lógica interna.

## Se conserva desde SGF

- sgf_id
- estado SGF como referencia
- grupo actual/remitente/creación como texto de origen
- observaciones
- RUT
- documento
- monto
- subtítulo
- número de traspaso/egreso si viene informado
- documentos recibidos
- raw_sgf_payload

## Se define internamente

- estado workflow
- tarea
- responsable
- unidad
- permisos
- checklist documental
- observaciones internas
- auditoría
- cierre
