# Tarea 03 — Seguridad, usuarios, roles y permisos

Implementar autenticación, usuarios, roles y permisos con Spatie Permission.

Criterios:

- Usuarios administrados por admin/superadmin.
- No auto-registro público.
- Policies/Gates para acciones sensibles.
- Permisos por módulo, workflow, documentos, reportes y administración.
